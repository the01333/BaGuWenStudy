（1）新建（New）

    刚创建的时候处于新建状态，还没有开始运行


（2）就绪（Runnable）

    当创建好的线程调用 start() 后，线程会进入就绪状态。
    这个状态下线程会等待线程调度器将其调度到CPU上执行


（3）运行（Running）

    当线程获得CPU时间片并执行 run() 的时候，线程进入运行状态，执行任务


（4）阻塞（Blocked）

    线程等待一个监视器锁（monitor lock）时进入阻塞状态，直到其他线程释放锁


（5）等待（Waiting）

    线程无期限地等待另一个线程显式唤醒它


（6）超时等待（Timed Waiting）

    线程等待另一个线程显示唤醒它，或者等待一定的时间后自动唤醒

（7）终止（Terminated）

    线程的 run() 执行完，或者抛出未捕获的异常
